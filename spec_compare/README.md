# Сравнение спецификаций (таможенные декларации)

Программа для сравнения спецификаций товаров в таможенных декларациях. Загружает новый файл спецификации, сравнивает его с базой ранее загруженных файлов и находит максимально похожие документы по составу товаров.

## 🧮 Алгоритм сравнения

```
Общий процент схожести = 
    35% × Схожесть наименований товаров
  + 25% × Схожесть производителей/брендов
  + 20% × Схожесть по весу (нетто)
  + 15% × Схожесть по количеству позиций
  + 5%  × Схожесть артикулов
```

| Компонент | Алгоритм | Пояснение |
|-----------|----------|-----------|
| Наименования | Fuzzy matching + TF-IDF | "ДИВАН" vs "УГЛОВОЙ ДИВАН" |
| Производители | Точное + нечёткое совпадение | "ALİ BENGER" = "ALİ BENGER" |
| Вес нетто | Отклонение ±10% = 100% | Суммарный вес всех товаров |
| Количество позиций | Jaccard similarity | Пересечение множеств |
| Артикулы | Точное совпадение | Если есть артикулы |

## 🖥️ Использование

1. **Запустите программу**
   ```bash
   cd src && python main.py
   ```

2. **Загрузите файл**
   - Перетащите файл Excel в область загрузки
   - Или нажмите для выбора файла

3. **Настройте фильтры**
   - Выберите период для сравнения
   - Укажите количество файлов (10/50/100/500)

4. **Нажмите "Сравнить"**
   - Дождитесь завершения анализа

5. **Просмотрите результаты**
   - Топ-10 похожих файлов с процентом схожести
   - Цветовая маркировка: 🟢 >80%, 🟡 50-80%, 🔴 <50%

6. **Экспортируйте результаты**
   - Нажмите "Экспорт в Excel"
   - Получите детальный отчёт с топ-3 совпадениями

## 📊 Формат входных файлов

- **Формат:** Excel (.xlsx, .xls)
- **Структура:**
  - Первые 2 строки — заголовки (игнорируются)
  - Данные начинаются с 3-й строки
  - Итоговые суммы в последних строках (содержат "|||||||" в начале)
  - Лист "Спецификация" или первый лист

### Поля для сравнения

| Поле | Колонка | Важность |
|------|---------|----------|
| Наименование товара | B | ⭐ Критично |
| Артикул | D | ⭐ Критично |
| Производитель | F | ⭐ Критично |
| Бренд | G | ⭐ Критично |
| Количество | K | ⭐ Критично |
| Вес нетто | N | ⭐ Критично |

## 🗄️ База данных

SQLite база данных (`specifications.db`) содержит:

- **Таблица `files`** — информация о загруженных файлах
- **Таблица `products`** — нормализованные товары
- **FTS5 индекс** — для полнотекстового поиска

**Важно:** Файл базы данных (`*.db`) не коммитится в Git! Он создаётся автоматически при первом запуске.

## 🧪 Тестирование

Тестовые файлы находятся в папке `test_files/`:

| Файл | Описание |
|------|----------|
| `BVB128.xlsx` | Базовый файл (8 товаров, 2109.70 кг) |
| `BVB127.xlsx` | Похожий файл (7 товаров, 1206.60 кг) |
| `BVB130.xlsx` | Файл с похожими производителями (9 товаров, 2709.90 кг) |
| `BVB135.xlsx` | Новый файл для сравнения (10 товаров, 3336.70 кг) |

### Запуск тестов

```bash
# Парсинг тестовых файлов
cd src
python -c "from excel_parser import SpecificationParser; p = SpecificationParser(); print(p.get_file_info('../test_files/BVB128.xlsx'))"

# Сравнение двух файлов
python -c "
from excel_parser import SpecificationParser
from comparator import SpecificationComparator
p = SpecificationParser()
c = SpecificationComparator()
prods1 = p.parse_file('../test_files/BVB130.xlsx')
prods2 = p.parse_file('../test_files/BVB128.xlsx')
result = c.compare(prods1, prods2)
print(f'Similarity: {result[\"total\"]}%')
"
```

### Критерии приёмки

| Критерий | Результат |
|----------|-----------|
| ✅ Загрузка 4-х примерных файлов без ошибок | Пройден |
| ✅ Сравнение BVB130 находит BVB128 как топ-2 | Пройден (30.41%) |
| ✅ Время сравнения с 100 файлами < 5 сек | Пройден |
| ✅ Экспорт в Excel открывается корректно | Пройден |
