# Specification Comparison Tool (Essenyurt Muhammad)

This repository contains a tool for comparing product specifications in customs declarations (Excel files).

## 📁 Project Structure

- `spec_compare/` - Main application source code and test files.
  - `src/` - Python source code (PyQt6).
  - `test_files/` - Example Excel files for testing.
- `venv/` - Python virtual environment (ignored by git).

## 🚀 Quick Start

1. **Activate Environment**:
   ```bash
   source venv/bin/activate
   ```

2. **Navigate to Source**:
   ```bash
   cd spec_compare/src
   ```

3. **Run Application**:
   ```bash
   python3 main.py
   ```
